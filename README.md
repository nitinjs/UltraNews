# UltraNews
Touch, Gestures &amp; Geo-location enabled News reader application which will allow user to read news from various sources over the web

a WCF project based on .NET 4.8<br/>
The technologies used to make this application are:<br/>
1. WCF<br/>
2. WPF<br/>
3. Windows SDK<br/>
4. SQL Lite<br/>
5. C#<br/>
6. Entity Framework<br/>
